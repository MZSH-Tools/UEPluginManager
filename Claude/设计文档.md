# UEPluginManager 设计文档

## 项目概述

UE 插件管理工具，用于统一管理 UE 项目及其关联引擎中的所有插件。

## 功能需求

### 项目加载
- [x] 读取 .uproject 文件
- [x] 解析引擎版本关联
- [x] 支持安装版引擎（通过注册表查找）
- [x] 支持源码版引擎（通过 GUID 查找）
- [x] 自动加载当前工作目录项目
- [x] 启动时检查项目有效性

### 插件扫描
- [x] 扫描项目 Plugins 目录
- [x] 扫描引擎 Plugins 目录
- [x] 解析 .uplugin 文件信息
- [x] 区分插件来源（项目/引擎）
- [x] 支持 UTF-8 BOM 编码文件
- [x] 支持尾随逗号的 JSON 格式

### 插件管理
- [x] 查看插件列表
- [x] 搜索插件（名称/描述/分类）
- [x] 查看插件详情
- [x] 查看依赖关系
- [x] 查看被依赖关系
- [x] 启用/禁用插件（修改 .uproject）
- [x] 打开插件目录
- [x] 打开项目目录
- [x] 打开引擎目录

### 界面
- [x] GUI 图形界面（PySide6）
- [x] 左右分栏布局
- [x] 工具栏显示项目目录和引擎目录
- [x] 插件列表树形显示（带搜索框）
- [x] 详情面板
- [x] 状态栏统计

## 待完善

- [ ] 批量启用/禁用
