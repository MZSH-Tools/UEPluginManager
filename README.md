# UEPluginManager

UE 插件管理工具，统一管理 UE 项目及其关联引擎中的所有插件。

## 功能

- **插件扫描** - 自动扫描项目、商城、引擎插件
- **分类管理** - 标签页分类显示（项目/商城/引擎）
- **引擎识别** - 支持安装版和源码版引擎
- **搜索** - 可选字段搜索（名称/作者/分类/描述/依赖/被依赖），实时显示各分类匹配数
- **排序** - 点击列标题按该列排序
- **依赖分析** - 查看插件依赖和被依赖关系
- **启用管理** - 直接启用/禁用插件（修改 .uproject），支持依赖连锁确认
- **插件移动** - 在项目和商城目录之间移动插件
- **插件删除** - 删除插件（移至回收站）
- **目录修正** - 修正乱码文件夹名为插件同名
- **快速访问** - 一键打开项目目录、引擎目录、插件目录
- **关闭项目** - 尝试关闭 UE 编辑器
- **重新加载** - 外部修改插件后可手动刷新

## 运行

```bash
# 创建环境
conda create -n UEPluginManager python=3.11 -y
conda activate UEPluginManager
pip install -r requirements.txt

# 启动
python Main.py
```

## 使用

1. 在 UE 项目目录下启动工具（自动加载当前目录项目）
2. 工具自动扫描项目和引擎插件
3. 使用搜索框定位插件
4. 点击插件查看详情
5. 勾选「在项目中启用」管理插件状态

## 注意事项

- **只读文件**：工具会自动移除只读属性（适用于 Perforce 等版本控制系统）
- **操作前建议**：关闭 UE 编辑器后再进行插件移动、删除、目录修正等操作
